{{ "1. What is an Earthquake?" }}

{{ "An earthquake occurs when tectonic plates beneath the Earth’s surface suddenly move. This releases energy and causes the ground to shake." }}

  • 🌋 {{ "Caused by movement of tectonic plates" }}
  • ⚡ {{ "Sudden release of built-up energy" }}
  • 🌍 {{ "Exact time cannot be predicted" }}

{{ "2. What You May Feel During an Earthquake" }}

  • {{ "Light shaking of the ground" }}
  • {{ "Furniture and windows vibrating" }}
  • {{ "Fans and hanging objects swinging" }}
  • {{ "Strong earthquakes may make standing difficult" }}

{{ "3. The Golden Rule: DROP – COVER – HOLD ON" }}

  • ⬇ {{ "DROP" }} – {{ "Drop to your knees to prevent falling" }}
  • 🛡 {{ "COVER" }} – {{ "Protect head and neck under a table or with arms" }}
  • ✊ {{ "HOLD ON" }} – {{ "Hold until shaking completely stops" }}

{{ "Never do this:" }}

  • ❌ {{ "Do not run outside during shaking" }}
  • ❌ {{ "Stay away from windows and glass" }}
  • ❌ {{ "Avoid cupboards, shelves, and ceiling fans" }}
  • ❌ {{ "Never use elevators" }}

{{ "4. What to Do in Different Situations" }}

{{ "Inside a building:" }}

  • {{ "Drop, Cover, and Hold On" }}
  • {{ "Stay inside until shaking stops" }}

{{ "In bed:" }}

  • {{ "Stay in bed and protect your head with a pillow" }}

{{ "Outside:" }}

  • {{ "Move to open ground" }}
  • {{ "Stay away from buildings and electric poles" }}

{{ "In a vehicle:" }}

  • {{ "Stop safely and remain inside the vehicle" }}
  • {{ "Avoid bridges, flyovers, and tunnels" }}

{{ "5. After the Earthquake" }}

  • {{ "Stay calm and check for injuries" }}
  • {{ "Be careful of aftershocks" }}
  • {{ "Avoid damaged buildings" }}
  • {{ "Follow school or community evacuation plans" }}
